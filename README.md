🎓 Smart Class Companion

Smart Class Companion is a web-based student productivity and collaboration platform designed to help students manage their academic activities efficiently. The application provides a centralized dashboard for tracking course progress, setting goals, managing tasks, sharing notes, and staying updated with class announcements.

Managing academic activities can be challenging when information is scattered across multiple platforms. Smart Class Companion brings everything together in a single, user-friendly dashboard, helping students stay organized and focused on their learning goals.

This project was developed as a capstone project using Python and Streamlit.

✨ Features

📊 Dashboard
- View academic progress at a glance
- Monitor course completion status
- Quick access to important information

🎯 Goal Management
- Set academic goals
- Track goal completion
- Monitor personal progress

✅ To-Do Tracker
- Create and manage tasks
- Mark tasks as completed
- Stay organized with daily activities

📢 Announcements
- Post and view class announcements
- Keep students informed about important updates

📁 Notes Sharing
- Upload and share study materials
- Access notes from a centralized location

👤 User Personalization
- Personalized experience for students
- Individual task and progress tracking

🎨 Enhanced User Interface
- Clean and interactive design
- Responsive and easy-to-use layout

🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- HTML/CSS
- Git & GitHub

⚙️ Installation

1. Clone the Repository

```bash
git clone https://github.com/TarunKumarShukla/Smart_Class_Companion-.git
```

2. Navigate to the Project Directory

```bash
cd Smart_Class_Companion-
```

3. Install Dependencies

```bash
pip install -r requirements.txt
```

4. Run the Application

```bash
streamlit run app.py
```
